# Frontend Design - Bharat AI Text Simplifier
## Complete UI/UX Specification

---

## Design Philosophy

### Principles
1. **Simplicity First**: Minimal clicks, maximum clarity
2. **Accessibility**: Usable by all age groups and literacy levels
3. **Responsive**: Works on all devices (320px to 4K)
4. **Fast**: Optimized for low-bandwidth connections
5. **Inclusive**: Supports multiple languages and scripts

---

## Color Palette

### Primary Colors
- **Primary Blue**: #2196F3 (Trust, technology)
- **Primary Dark**: #1976D2 (Headers, emphasis)
- **Primary Light**: #BBDEFB (Backgrounds, highlights)

### Secondary Colors
- **Success Green**: #4CAF50 (Success messages)
- **Warning Orange**: #FF9800 (Warnings)
- **Error Red**: #F44336 (Errors)
- **Neutral Gray**: #757575 (Secondary text)

### Background Colors
- **White**: #FFFFFF (Main background)
- **Light Gray**: #F5F5F5 (Input areas)
- **Dark Gray**: #212121 (Footer)

---

## Typography

### Font Family
- **Primary**: 'Segoe UI', 'Roboto', 'Arial', sans-serif
- **Hindi**: 'Noto Sans Devanagari', sans-serif
- **Monospace**: 'Courier New', monospace (for code)

### Font Sizes
- **H1 (Main Title)**: 32px (mobile: 24px)
- **H2 (Section Headers)**: 24px (mobile: 20px)
- **Body Text**: 16px
- **Small Text**: 14px
- **Button Text**: 16px (bold)

### Line Heights
- **Headers**: 1.2
- **Body**: 1.6
- **Buttons**: 1.4

---

## Layout Structure

### Grid System
- **Desktop**: 12-column grid, max-width 1200px
- **Tablet**: 8-column grid, max-width 768px
- **Mobile**: 4-column grid, max-width 480px

### Spacing Scale
- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **xxl**: 48px
